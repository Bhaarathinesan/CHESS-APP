# ChessArena PWA Screenshots

This directory contains screenshots for the PWA installation prompt and app store listings.

## Required Screenshots

- `desktop-1.png` - 1280x720px (16:9 aspect ratio)
  - Show the main chess board interface
  - Include game controls and move list
  - Demonstrate the clean, modern UI

### Mobile/Narrow Form Factor
- `mobile-1.png` - 750x1334px (iPhone 6/7/8 size)

## Guidelines

- Use actual application screenshots (not mockups)
- Show the app in use with realistic content
- Ensure good lighting and contrast
- Avoid including personal information
- Use the dark theme (default) for consistency

## Capturing Screenshots

### Desktop
1. Open the app in Chrome at 1280x720 resolution
2. Navigate to an active game or demo page
3. Use Chrome DevTools > Device Toolbar for consistent sizing
4. Take screenshot: Cmd+Shift+4 (Mac) or Snipping Tool (Windows)

### Mobile
1. Use Chrome DevTools > Device Toolbar
2. Select iPhone 6/7/8 (375x667) or similar
3. Zoom to 200% for retina display
4. Capture at 750x1334px resolution
5. Or use actual device screenshots
